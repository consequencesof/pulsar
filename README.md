# Pulsar

Мессенджер в реальном времени с примитивным дизайном 

## Возможности

- Регистрация и авторизация (@username + nickname)
- Личные сообщения в реальном времени
- Поиск других пользователей по @username
- Аватарки (загрузка с устройства)
- Смена темы оформления (Aurora / Dusk / Ember)  временно не ворк
- Режим Liquid Glass (как у Apple)
- Снегопад с настройками интенсивности
- Просмотр профилей (основной @, NFT @, анонимный номер)
- Редактирование профиля
- Удаление чатов (только у себя или у всех)
## Технологии

- HTML / CSS / JavaScript (чистый фронтенд)
- TailwindCSS
- Firebase Realtime Database (бэкенд)
- GitHub Pages (хостинг)
